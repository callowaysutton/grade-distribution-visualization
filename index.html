Introduction

Created by Calloway Sutton


This website exists for three main reasons. The first, I was mad that at IU an A- isn't equal to a A+ as it unfairly skews the grade distribution down. For any other letter grade, if you get the minus variant, say a B-, you could work extra hard in an equivalent credit hour class and balance that towards getting a B average. However, if you get an A- or below in a single class it is impossible to get that semesters grade back up to a 4.0. So this visualization was going to be a two-fold demonstration as to how changing the policy to where an A- equalling a 4.0 would do little to artificially inflate ~80% of students but help fix the skew for the top 20%

The second reason I made this website was to learn PlotlyJS and D3.js, two visualization frameworks, along with Julia. I saw Julia being used in various places and thought it looked interesting, even though it's mostly used for Data Science applications, and it's actually been a blast to use. I usually main using Python with PyO3 for performance critical code, but with Julia it's just straight up unnecessary! You can leave things abstract for the price of performance, but if you need to optimize performance from what I've seen it's only around 25%, or less, worse than my equivalent C code. I also wanted to go out and make a website that was dynamic and interactive to show my friend that you can do some amazing things in just pure client side code. I really can't wait to make another project with Julia and I hope it gets more popular in the future as the language is just so nice to use.

The third and final reason for this tool existing is to provide a fun way of looking at the different classes provided by IU as well as giving a glimpse into what other major's courseloads could be like.

This tool is mainly for visualizing the past semester's grade distribution across the campus, but let me know if you think something even more useful could be made from this idea! You can contact me via the handle @callowaysutton basically anywhere.
